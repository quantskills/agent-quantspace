# Reports

[中文说明](README-zh.md)

This directory is for local generated research outputs.

Do not commit private reports, backtest result dumps, generated charts, or
research logs to the public repository.

The `strategy_examples/` subdirectory is the exception: it contains sanitized
public Markdown reports and performance PNGs generated from existing daily
Parquet files under `data/market/1d/`.

## Intended Use

- HTML/PDF reports generated by `skills.report`.
- Local charts, tables, and diagnostics created during experiments.
- Temporary artifacts for notebooks or scripts.
- Public example strategy Markdown reports and PNG charts under `strategy_examples/`.

Keep this directory clean in public commits. If an output is needed as a test
fixture or documentation asset, place a small sanitized sample in the relevant
test or docs directory instead.
