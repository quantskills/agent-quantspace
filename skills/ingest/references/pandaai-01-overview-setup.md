<h3>下面提供 PandaData Python SDK（Pro会员版本）完整使用指南，包含环境准备、安装、初始化、数据调取及结果示例。</h3>

**一、前提条件** <br/>
使用PandaData数据调取功能前，需完成以下准备：
1. 已在 **PandaAI官网** 完成用户注册
2. 已获取 **PandaAI token凭证**

**二、Python SDK 环境准备**
1. 基础环境要求
已安装 **Python 3.10+**（推荐版本）
推荐安装 **Anaconda** 集成开发环境，并正确配置环境变量

**三、SDK 下载与安装**

### 方式1：pip 命令安装（推荐）
```bash
# 常规安装
pip install panda_data

# 网络超时，使用国内pip源安装
pip install panda_data -i https://pypi.tuna.tsinghua.edu.cn/simple
```
### 方式2：源码包安装
1. 访问官网下载源码包：[https://pypi.python.org/pypi/panda_data/](https://pypi.python.org/pypi/panda_data/)
2. 解压后执行安装命令：
```bash
python setup.py install
```
### 版本升级
```bash
pip install panda_data --upgrade
```

### 查看当前安装版本
```python
import panda_data
print(panda_data.__version__)
```

**四、SDK 初始化与Token配置**

### 1. 导入依赖包
```python
import panda_data
```

### 2. 初始化 Token 认证
> 账号规则：`86+官网注册手机号`（示例：8617777777777）
> 密码与PandaAI官网密码保持一致

```python
if __name__ == "__main__":
    # 初始化token认证
    panda_data.init_token(
        username="你的86开头手机号",
        password="你的官网密码"
    )
```

**五、股票数据调取示例**

### 完整调用代码（基础版）
```python
import panda_data

if __name__ == "__main__":
    # 1. 初始化token
    panda_data.init_token(
        username="你的86开头手机号",
        password="你的官网密码"
    )
    
    # 调取市场数据
    result = panda_data.get_market_data(
        symbol="000001.SZ",
        start_date="20200101",
        end_date="20250101",
        type="stock"
    )
    
    print(result)
```

### 单只股票日线数据调取（精简示例）
```python
import panda_data

# 初始化token
panda_data.init_token(username="你的账号", password="你的密码")

# 获取000001.SZ 2025年1月日线数据
result = panda_data.get_market_data(
    symbol=["000001.SZ"],
    start_date="20250101",
    end_date="20250131",
    type="stock",
    fields=[],
    indicator="000300",
    st=True
)

print(result)
```
