# Docs

[English README](README.md)

这个目录存放 QuantSpace 的项目级文档。

## 文件说明

- `architecture.md`：轻核心架构和公开策略流程。
- `data_layout.md`：本地数据布局和 pool schema。
- `examples.md`：可运行 demo 命令。
- `panda_data_ingest.md`：PandaData 设置、符号转换和导入用法。
- `open_source_scope.md`：开源仓库应该包含和排除的内容。
- `private_extension_pattern.md`：推荐的私有仓库扩展方式。
- `project-skills.md`：公开 skill 注册表。

文档需要和当前公开边界保持一致。不要写入私有策略名称、私有数据路径、已删除模块或未公开 adapter。
