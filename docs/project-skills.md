# Project Skills

| Skill | Purpose |
|-------|---------|
| `ingest` | PandaData/PandaAI data access and symbol conversion |
| `store` | Parquet storage, explicit symbol panels, pools, factors, backtests, model metadata |
| `compute` | Indicators, features, public labels |
| `analyze` | Vectorized backtests, IC, grouped returns, attribution helpers, tearsheets |
| `construct` | Weighting, filters, strategy combining |
| `model` | Optional ML model helpers |
| `research` | Factor screening, parameter sweeps, strategy comparison |
| `report` | HTML/Markdown reports and charts |

Each skill directory contains a `SKILL.md` operating guide.
